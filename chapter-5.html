Chapter Five: Predictive Data Analysis

1. What is predictive data analysis?

Predictive analysis focuses on forecasting future events using data.

2. Predictive analysis mainly uses:

It relies on past and current data to predict the future.

3. Predictive analysis is most similar to:

Like weather forecasting, predictions are probabilistic.

4. Predictions are:

Predictive results are probabilities, not certainties.

5. Which is NOT a goal of predictive analysis?

Visualization is supportive, not a main goal.

6. Predicting future sales is an example of:

Forecasting predicts numerical future values.

7. Spam vs Not Spam is an example of:

Classification predicts categories.

8. Predicting loan default probability is an example of:

It estimates risk of an event.

9. What is the first step in predictive analysis workflow?

The goal must be clearly defined first.

10. Poor data preparation usually leads to:

Model quality depends on data quality.

11. Which step involves understanding patterns and relationships?

EDA explores data before modeling.

12. Which step chooses the best algorithm?

Model selection determines which method to use.

13. Splitting data into training and test sets helps to:

It checks generalization ability.

14. Which step measures how good a model is?

Evaluation uses performance metrics.

15. Using a model in real-world applications is called:

Deployment applies the model practically.

16. Which is a statistical method for prediction?

Regression is a statistical technique.

17. Linear regression is mainly used to predict:

Linear regression predicts continuous values.

18. Logistic regression predicts:

Logistic regression is for classification.

19. Time series analysis is used when data:

Time series data includes a time dimension.

20. Which is an example of time-series prediction?

Sales over time form a time series.

21. Decision Trees work based on:

Trees split data using decision rules.

22. Random Forest improves accuracy by:

It aggregates many decision trees.

23. Which model is best for complex patterns like images?

Neural networks handle complex data.

24. Ensemble methods mean:

Ensemble combines models for better performance.

25. Which is an ensemble technique?

Boosting is an ensemble method.

26. Bayesian methods are based on:

Bayesian models use probability theory.

27. Markov Chains predict:

They depend on the current state.

28. Predictive analysis is widely used in business to:

Businesses use predictions for strategy.

29. Fraud detection is common in:

Financial systems detect fraud using prediction.

30. Predicting disease risk is an application in:

Healthcare uses predictive models extensively.

31. Which metric is used for classification models?

Accuracy evaluates classification.

32. Precision measures:

Precision focuses on predicted positives.

33. Recall measures:

Recall shows how many positives were found.

34. Which metric is used for regression?

RMSE measures regression error.

35. Lower RMSE means:

Lower error indicates better predictions.

36. Which value indicates how well regression fits data?

R² shows goodness of fit.

37. Which language is widely used for predictive analysis?

Python has powerful ML libraries.

38. Which Python library is commonly used for ML?

Scikit-learn supports predictive models.

39. Overfitting means:

Overfitting harms generalization.

40. Which practice helps reduce overfitting?

Validation checks generalization.

41. Interpretability means:

Interpretability explains predictions.

42. Ethical concerns in predictive analysis include:

Bias can lead to unfair predictions.

43. Predictive models should be:

Models need monitoring over time.

44. Predictive analysis combines:

It uses both statistics and ML.

45. Which statement is TRUE?

Predictive analysis supports, not replaces, decisions.
