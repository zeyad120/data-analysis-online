Chapter Six: Data Analysis Tools & Software

Python is popular for data analysis and ML.

Power BI is designed for dashboards.

16. Which tool does NOT require programming knowledge?

17. Which tool is essential for any data analyst?

SQL is a core skill for data analysts.

18. Which library is used for data visualization in Python?

19. Tools selection depends mainly on:

Different problems need different tools.

SQL is optimized for queries.

Python is open-source.

Big data tools handle massive datasets.

23. Which tool is best for statistical research?

R is widely used in statistics.

24. Which tool combines visualization and reporting?

25. There is one tool that fits all data analysis tasks.

Different tasks require different tools.
