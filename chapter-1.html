Chapter One: Foundations

Understand how we collect, clean, and prep data before any analysis.

1. What is the primary goal of data analysis?

Data analysis focuses on extracting meaning and insights from raw data, not just collecting or storing it.

2. Why is data often referred to as “the new oil”?

Raw data has little value unless processed and analyzed, just like crude oil.

3. Which step comes immediately before modeling in the data analysis process?

EDA helps understand patterns and anomalies before applying models.

4. Customer satisfaction levels (Poor, Good, Excellent) represent which data type?

The categories have order but no fixed numerical distance.

5. Which scale of measurement has a true zero?

Ratio scales allow meaningful comparisons like “twice as much”.

6. Eye color is an example of:

Eye color is categorical with no inherent order.

7. Predicting future loan default risk is an example of:

It uses historical data to forecast future outcomes.

8. Why is data cleaning essential?

Clean data reduces errors and misleading insights.

9. Which tool is best suited for big data processing?

Hadoop is designed for distributed large-scale data.

10. Which challenge is mainly ethical rather than technical?

Privacy concerns involve legal and ethical responsibility.

11. Temperature measured in Celsius uses which scale?

Zero does not represent absence of temperature.

12. Detecting unusual credit card spending is an example of:

It predicts potential fraud behavior.

13. Which step converts numerical results into real-world meaning?

Interpretation explains what results actually mean.

14. Height measured in centimeters is classified as:

Height has a true zero and meaningful ratios.

15. Improving athlete training using performance data is an example of:

Optimization uses data to improve outcomes.

16. Which is NOT quantitative data?

Ordinal data is qualitative.

17. In the cooking analogy, data cleaning is like:

Cleaning prepares data before analysis.

18. Data without analysis is considered useless because it lacks:

Insights come only from interpretation.

19. Census data is mainly used in which domain?

Census data supports government planning.

20. Which statement about interval data is correct?

Interval data allows comparison of differences but not ratios.

21. Which statement best distinguishes data analysis from data collection?

Data analysis transforms raw data into information and insights, while data collection only gathers the data.

22. Which of the following best describes raw data?

Raw data is data in its original form before cleaning or analysis.

23. Why is data analysis critical in decision-making?

Data analysis supports decisions by providing evidence, not guarantees.

24. Which of the following best explains the phrase "Data is the new oil"?

Like oil, data must be processed and analyzed to generate value.

25. Which step of the data analysis process focuses on finding errors and inconsistencies?

Data cleaning removes errors, duplicates, and missing values.

26. Which data type allows mathematical operations like addition and multiplication?

Quantitative data is numerical and supports arithmetic operations.

27. Which scale of measurement has no true zero?

Interval scales (e.g., Celsius) lack a true zero.

28. Which of the following is an example of ordinal data?

Ordinal data has a meaningful order but no fixed intervals.

29. Which role primarily focuses on building predictive models?

Data scientists specialize in modeling and prediction.

30. What is the main drawback of big data?

Big data introduces challenges in storage, processing, and analysis.

31. Which application of data analysis focuses on early disease detection?

Healthcare uses data analysis to detect diseases early.

32. Which step comes immediately after data collection?

Cleaning prepares collected data for analysis.

33. Which data type is gender?

Gender is categorical without order.

34. Which is NOT a challenge in data analysis?

Visualization is a tool, not a challenge.

35. Why is bias dangerous in data analysis?

Bias distorts results and conclusions.

36. Which statement about qualitative data is true?

Qualitative data describes qualities or attributes.

37. Which field uses data analysis to detect fraud?

Banks use data analysis to detect fraudulent activities.

38. Which output is produced at the reporting stage?

Reporting communicates insights and findings.

39. Which term refers to discovering trends over time?

Time-series analysis examines trends over time.

40. Which tool is mainly used for big data processing?

Hadoop handles large-scale distributed data.

41. Which of the following is discrete data?

Discrete data consists of countable values.

42. Which is an example of ratio scale data?

Salary has a true zero and fixed intervals.

43. What is the main purpose of data visualization?

Visualization helps communicate patterns and insights.
