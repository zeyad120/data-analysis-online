Chapter Four: Statistical Analysis

1. What is the main goal of statistical analysis?

Statistical analysis focuses on understanding and interpreting data to aid decision-making.

2. Which type of statistics summarizes data without making predictions?

Descriptive statistics describe and summarize datasets.

3. Which measure describes the center of a dataset?

The mean is a measure of central tendency.

4. Which measure is least affected by extreme values?

The median is resistant to outliers.

5. Which statistic measures variability in data?

Standard deviation measures how spread out data values are.

6. Which type of statistics allows conclusions about a population using a sample?

Inferential statistics generalize sample results to populations.

7. What is a population in statistics?

Population refers to the whole group being studied.

8. What is a sample?

A sample is part of the population used for analysis.

9. Which level of measurement is gender?

Gender is categorical with no order.

10. Which level of measurement has a true zero?

Ratio data has a meaningful zero.

11. Temperature measured in Celsius is an example of:

Zero in Celsius does not mean absence of temperature.

12. Which measure shows the difference between maximum and minimum values?

Range = Max − Min.

13. Variance is best described as:

Variance is the average squared distance from the mean.

14. Standard deviation is:

SD is the square root of variance.

15. What does a small standard deviation indicate?

Small SD means values are close to the mean.

16. Which statistic is best for skewed data?

Median is robust in skewed distributions.

17. What is inferential statistics mainly based on?

Inferential statistics use samples to infer about populations.

18. What is a hypothesis?

Hypotheses are assumptions tested statistically.

19. What does the null hypothesis (H₀) represent?

H₀ assumes no effect or difference.

20. Which value is commonly used as a significance level (α)?

0.05 is the standard significance level.

21. Rejecting the null hypothesis means:

Rejecting H₀ means evidence supports the alternative hypothesis.

22. What does a confidence interval represent?

CI gives a plausible range for population values.

23. A 95% confidence interval means:

CI reflects confidence, not certainty.

24. Which analysis studies relationship between variables?

Correlation measures strength and direction of relationships.

25. Correlation coefficient values range between:

Correlation ranges from -1 to +1.

26. Correlation of -1 indicates:

-1 is perfect negative correlation.

27. Correlation does NOT imply:

Correlation alone cannot prove cause-effect.

28. Regression analysis is mainly used for:

Regression predicts dependent variables.

29. In regression, the variable being predicted is:

Dependent variable is the outcome.

30. Which regression uses one independent variable?

Simple linear regression uses one predictor.

31. What does ANOVA test?

ANOVA compares means of 3 or more groups.

32. ANOVA is an example of:

ANOVA draws conclusions about populations.

33. Which chart best shows data distribution?

Histograms show frequency distribution.

34. Which plot best detects outliers?

Boxplots highlight outliers.

35. Scatter plots are mainly used to show:

Scatter plots show relationships.

36. Which mistake is common in statistical analysis?

Correlation ≠ causation is a major pitfall.

37. Which factor most affects inferential accuracy?

Larger samples usually give more reliable results.

38. Which statistic compares variability across datasets?

CV compares relative variability.

39. Which distribution is symmetric?

Normal distribution is symmetric.

40. What does skewness describe?

Skewness measures asymmetry.

41. Which step comes first in hypothesis testing?

Hypotheses must be defined first.

42. Which value indicates strong evidence against H₀?

Small p-values suggest rejecting H₀.

43. Statistical analysis is MOST important for:

Statistics supports evidence-based decisions.

44. Which field heavily relies on statistical analysis?

Healthcare uses statistics extensively.

45. Which statement is TRUE?

Statistics reduces uncertainty but never guarantees certainty.
